# 共享车位出租平台 — 项目报价单

## 项目概述

基于 Boot 停车开放平台 API，构建共享车位出租平台，包含租客微信小程序和 Web 管理后台。

## 技术栈

| 层级 | 技术选型 |
|------|---------|
| 后端 | RuoYi-Vue（Spring Boot + MyBatis，自带 RBAC/代码生成/日志） |
| 小程序 | uni-app |
| 管理后台 | RuoYi 内置（Vue3 + Element Plus） |
| 数据库 | MySQL |
| 缓存/队列 | Redis |

### 工期估算

按最大并行度（后端、前端、设计同时推进），关键路径为后端开发（62 天）。考虑到前后端联调依赖和实际并行效率：

| 阶段 | 周期 | 说明 |
|------|------|------|
| UI 设计 + 基础架构 | 第 1-3 周 | 小程序 UI 设计 + RuoYi 基座搭建 + Boot 对接 + uni-app 初始化 |
| 核心开发 | 第 3-9 周 | 前后端并行开发小程序 + 后台（RuoYi 代码生成加速） |
| 联调测试 | 第 9-11 周 | 前后端联调 + Boot 联调 + 集成测试 |
| 部署上线 | 第 11-12 周 | 部署 + 小程序审核 + 上线观察 |
| **总工期** | **约 12 周（3 个月）** | |
