# 共享车位出租平台 — 项目报价单

## 项目概述

基于 Boot 停车开放平台 API，构建共享车位出租平台，包含租客微信小程序和 Web 管理后台。

## 技术栈

| 层级 | 技术选型 |
|------|---------|
| 后端 | Java Spring Boot |
| 小程序 | uni-app |
| 管理后台 | Vue3 + Element Plus |
| 数据库 | MySQL |
| 缓存/队列 | Redis |

### 工期估算

| 阶段 | 周期 | 说明 |
|------|------|------|
| 需求/设计 | 第 1-2 周 | PM 需求细化 + UI 设计并行 |
| 基础架构 + 设计 | 第 2-4 周 | 后端脚手架/数据库/Boot 对接 + UI 设计 + 前端脚手架 |
| 核心开发 | 第 4-10 周 | 前后端并行开发小程序 + 后台 + 业务逻辑 |
| 联调测试 | 第 10-12 周 | 前后端联调 + Boot 联调 + 集成测试 |
| 部署上线 | 第 12-13 周 | 部署 + 小程序审核 + 上线观察 |

## 备注

1. 以上报价基于已确定的需求范围（详见 `CONTEXT.md` 和 `docs/adr/`），需求变更需另行评估
2. 不含服务器、域名、SSL 证书、微信小程序认证（300 元/年）、微信支付商户号等第三方费用
3. 不含 iOS/Android App 开发，仅含微信小程序
4. 不含第二家停车系统（捷顺/科拓）的适配器实现，也不含适配器抽象层（推迟到二期）
5. 上线后维护/运营支持需另行签订维保合同
6. 结算为简化版（Excel 导出 + 手动结算），自动结算引擎推迟到二期
7. 数据大盘推迟到二期
